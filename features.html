Features

Arrays

  • All arrays are actually objects. Inherit from Array base class.

Constructors

Constructors in C# are similar to constructors in Java. 

// An example of a constructor declaration.
public Point() {

}

// An example of a constructor invocation.
Point myPoint = new Point();

Destructors

  • Special methods used to destruct instances of classes.
  • Invoked when the garbage collector determines that the object is no longer being used. 
// An example of a destructor declaration.
~ Point() {
    // Cleanup statements.
}
