Description

C# is an object-oriented language from Microsoft aimed at combining the power of C++ and Java. It is specifically designed to work with the Microsoft .NET platform.

Microsoft describes it as an elegant and type-safe object-oriented language that enables developers the ability to build a variety of secure and robust applications based on the C# language and the .NET framework.

The Language

Microsoft introduces the C# syntax as highly expressive yet easy to learn. This is mostly due to the fact that looks and acts so much like Java. It does however have features that are not available in Java such as "nullable value types, enumerations, delegates, lambda expressions and direct memory access."

C# supports generic methods and types, which provide increased type safety and performance, and iteraters, which enable implementers of collection classes to defin custom iteration behaviors that are simple to use by client code.

As an object-oriented language, C# supports the concepts of encapsulation, inheritance, and plymorphism. With it's basic object oriented principles, C# makes it easy to develop software components through several innovative language constructs.

Advantages

  • Automatic garbage collection
  • Pointers are not required
  • Declaration of functions and classes are not needed
  • Classes can be defined within other classes
  • No global functions or variables, everything belongs to a class
  • All variables are automatically initialized to their default values but can be initialized manually using static constructors

Programming

C# can be written in any text editor or IDE but Microsofts Visual Studio natively supports C#.

C# Programs can be compiled and ran from within Visual Studio.

C# is now fully supported on the Linux platform as well and can be compiled through the Make Utility.

C# uses the .NET Framework Class Library making it very flexible and easy to use.

On other hand, C# heavily relies on the .NET Framework.
