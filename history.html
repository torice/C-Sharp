Background of C#

Highlights

  • - Microsoft created C# in late 90s as primary language for .NET platform
  • - Chief architect: Anders Hejlsberg
  • --designed Turbo Pascal, Borland Delphi, and Visual J++
  • - Official Release: July, 2000
  • - Standardized by ECMA and ISO
  • - Supporters: Hewlett-Packard, Intel, and Microsoft
  • - Directly related to C, C++, Java
  • - Name taken from C and C++ languages: C# is a 'semitone higher'

Classification

A multi-paradigm programming language:

- Imperative

- Declarative

- Functional

- Generic

- Strongly typed

- Object Oriented

- Component Oriented

Creation of C#

- Sun wouldn't allow Microsoft to make changes to Java

- Microsoft tried to extend Java in Visual J++ to tie it more closely to the Windows OS

- Sun hit Microsoft with a lawsuit for violating the terms of its licensing agreement

- Microsoft dumped its Visual J++ product and its developers and came up with idea C#

- Around 1997, Microsoft started a project internally known as Project Lightning

- Codename of C# was 'Project Cool', a 'clean-room' implementation of Java

Context and Influencers

- C# directly descended from two of the most successful computer languages: C and C++

- From C, it derives its syntax, many of its keywords, and most of its operators

- It builds upon and improves the object model defined by C++

- C invented in 70s: most widely used structured programming language of the 1980s

- C was unmanageable for large programming projects

- C++ (1979) Bjarne Stroustrup: most widely used programming language by 2000

- Java (1991) Sun Microsystems and James Gosling; syntax and philosophy derived from C++; innovations driven by changes in computing environment: the Internet and need for portability across connected machines

- Solved problem of portability (rise of Internet); create cross-platform, portable code = rapid acceptance

Evolution of C#

C# has rapidly evolved since intial release in 2000:

V1.0: (2000) Modern, Object Oriented, Simple, Flexible, Typesafe, Managed, Garbage Collection, Cross-platform

V2.0: (2005) Generics, Anonymous Method, Partial Class, Nullable Type

V3.0: (2008) LINQ (Language Integrated Query), Lamda Expression, Extension Method, Anonymous Type, Var

V4.0: (2010) Named and Optional Parameters, Dynamic Binding

V5.0: (2012) Async Programming
