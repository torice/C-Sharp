Conventions

  • Method names use unique variant of camel case, sometimes referred to as UpperCamelCase. 
    // C# method call.
object.MethodCall();

// Java method call.
object.methodCall();

Variable Declaration

  • C# is both strongly and weakly typed.
  • Variables may be explicitly typed: 
    string sentence = "Hey dare.";
  • Variables may also be implicitly typed: 
    var sentence = "Hey dare.";
  • C# conventions suggest the use of 'var' type only when type of object is obvious from the right hand side of assignment. 
    // Recommended.
var sentence = "This is clearly a string.";
var number = 27;

int var4 = Example.ThisCouldReturnAnything();
